# Day 7 - 同値・前後関係とアノテーション

Day 7では、Java Gold試験の出題頻度は少ないものの、実務の観点でぜひ習得しておきたい事柄を紹介する。

* アノテーション - @Overrideや@Deprecatedなど、クラスやメソッドなどに注釈を付けることができる。
